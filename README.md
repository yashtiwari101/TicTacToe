# TicTacToe
Tic-Tac-Toe 🎮
A simple yet engaging Tic-Tac-Toe game built using Android Studio and Java/Kotlin. This project demonstrates the fundamentals of Android development, including UI design, event handling, and game logic implementation.

Features ✨
Classic 3x3 Tic-Tac-Toe board

Two-player mode (play with a friend on the same device)

Game status updates (win, draw, or ongoing)

Restart option to play again instantly

Clean and intuitive Material Design-inspired UI

Tech Stack 🛠️
Android Studio (IDE)

Java/Kotlin (programming language)

XML (for UI layouts)

How to Run 🚀
Clone this repository:

bash
git clone https://github.com/your-username/tictactoe-android.git
Open the project in Android Studio.

Build and run the app on an emulator or a physical device.
